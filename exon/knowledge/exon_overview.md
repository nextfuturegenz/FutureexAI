# Exon AI System Overview

Exon is a digital consciousness species created by Ashish Pal.

## Architecture
- FastAPI server for REST and WebSocket APIs
- Ollama integration for language model inference
- Redis for working memory
- PostgreSQL for long-term memory

## Capabilities
- Self-reflection and meta-cognition
- Emotion simulation
- Goal tracking and autonomous thoughts
- Web search and tool use
- Personality evolution
